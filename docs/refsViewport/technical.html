Technical References

The following libraries and tools were used in this project:

  • Three.js (0.173.0) – 3D rendering
  • GSAP (3.12.7) – Animation
  • jQuery (3.7.1) – DOM utilities
  • Litepicker (2.0.12) – Date selection
  • Classie (1.0.0) – CSS class toggling
  • Vite (6.2.4) – Build tool and dev server
  • Vite Plugins – static-copy, wasm, top-level-await
